# Pré-Requisitos
- Compilador de C++ (ex: gcc)
- bash

# Inicializar Programa:
Basta utilizar:
~~~ bash
g++ main.cpp Grafo.cpp Guloso.cpp -o run -O3
~~~
e depois
~~~ bash
./run nomedainstancia p q
~~~

# Rodar Testes:
Basta rodar o arquivo executável __run_teste.sh__ na pasta principal (/trabalho-grafos) com:
~~~ bash
./run_teste.sh
~~~
